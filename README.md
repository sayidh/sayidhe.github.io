# How to set up the project

## 1. Assets

Yarn
- https://yarnpkg.com/

Gulp 4.0.0 Info
- https://fettblog.eu/gulp-4-parallel-and-series/

Google Fonts
- https://fonts.google.com


## 2. Set up environment

Install yarn
```
brew install yarn
```

Install packages
```
yarn install
```

### 3. Run the project

Run the gulp for development
```
gulp serve
```

Run the gulp for production, will compress the css file
```
gulp production
```
